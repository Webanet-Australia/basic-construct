  • Category

    Define category, this is used by the navigation on the shop page. A product can belong to many categories. The home category is used to add products to the Carousel.

  • Price

    Set the price of the product. Other Woocomerce product options can be also be defined.

    A Bootstrap Card is used to display product details on the home, shop and region pages.
