  • Hero Image Carousel.

    The Hero image is the full size image at the top of the page. Adding multiple images creates a carousel effect and scrolls through the list of images displaying each for a brief time.

    Open For Editing

    • The overlayed text is the main editor content.

  • Region Tabs

    Add tabs for the most popular or recently added regions.

    Open Tabs
    • Tab Parts

      Each tab is made up of these parts

    • Name and Main Copy

      Name the tab and add the main copy text.

    • Lead Copy

      Lead Copy is the tab's first paragraph

    • Link

      Select a page from the drop down to link to from the tab. Enter a caption for the link.

    • Extra Fields

      The extra fields not previously mentioned define some further settings for customizing the tabs, the field values should be the same for each tab. If you want to create additional tabs it's probably easiest to copy the values from one of the existing tabs.
